# ST Card Manager

一个现代化的SillyTavern角色卡管理工具，支持PNG和JSON格式的角色卡导入、管理、编辑和导出。

## 功能特性

- 📁 支持PNG和JSON角色卡导入
- 🏷️ 标签和文件夹管理
- 🔍 强大的搜索和筛选功能
- 🎨 亮色/暗色主题切换
- 📊 重复卡片检测
- 💾 IndexedDB本地存储
- 📦 批量导出为ZIP
- 🖼️ 图片预览和编辑
